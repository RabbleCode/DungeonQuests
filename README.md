# DungeonQuests

A simple WoW: Classic (including Season of Discovery) addon to track acquisition and completion of dungeon quests

Usage:

- `/dungeonquests all` or `/dq all` - Updates and displays a summary of all dungeon quests up through Maraudon (more to come)
   <img width="786" height="209" alt="image" src="https://github.com/user-attachments/assets/9c559b2b-999a-40d4-aae8-6ce879f2c51a" />

- `/dungeonquests <alias>` or `/dq <alias>` - Updates and displays progress for the specified dungeon alias (see below)
   <img width="522" height="169" alt="image" src="https://github.com/user-attachments/assets/980d4b05-78f6-43c5-aed3-1e7ab97a0302" />


Dungeon aliases:

- `RFC` - Ragefire Chasm
- `VC` - Deadmines
- `WC` - Wailing Caverns
- `BFD` - Blackfathom Deeps
- `SFK` - Shadowfang Keep
- `RFK` - Razorfen Kraul
- `STOCK` - The Stockade
- `RFD` - Razorfen Downs
- `SM` - Scarlet Monastery
- `ULDA` - Uldaman
- `ZF` - Zul'Farrak
- `MARA` - Maraudon
- `BRD` - Blackrock Depths
